Data Science: natural language processing & network inference

The goal of this project is to analyse chat data with my girlfriend; apply statistical methods, graph theory, and other data science techniques.

Please note that this project is presented in knitted interactive .html reports, you can view these below or by downloading them from the github repo at the directory ./reports/ *or* you can visit them hosted on my website at: derecksnotes.com: ds-portfolio.
